# Contributing

Thank you for your interest in contributing to this project!

Currently, this project is developed and maintained by a single author.
External contributions are not actively accepted at this time.

## Guidelines

If you still wish to contribute:

* Fork the repository
* Create a new branch (`feature/your-feature-name`)
* Commit your changes clearly
* Open a Pull Request with a detailed description

## Notes

* Keep code clean and readable
* Follow the existing coding style
* Write meaningful commit messages

## Contact

If you have suggestions or ideas, feel free to open an issue.
